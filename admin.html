⚙️ Configurar Google Sheets

Cole aqui a URL do seu Google Apps Script Web App para sincronizar agendamentos automaticamente.
Como configurar? →

Como configurar o Google Sheets:
1. Acesse sheets.google.com e crie uma planilha.
2. Vá em Extensões → Apps Script.
3. Cole o código abaixo e clique em Implantar → Novo Implantação → App da Web.
4. Defina acesso como "Qualquer pessoa" e copie a URL gerada aqui.

function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ data.nomeCliente, data.celular, data.data, data.hora, data.cafe, data.brinde, data.obs, data.criadoEm ]); return ContentService.createTextOutput("ok"); }
